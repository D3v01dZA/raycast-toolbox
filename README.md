# Toolbox

A Raycast extension with developer utilities. Type into the search bar and get instant results.

## Commands

### Convert

Quick unit conversions, number base encoding, color conversion, and date/time parsing. Press **Cmd+H** to see all supported categories with examples.

Press **Enter** on any result to copy it to the clipboard.

### Networking

Subnet calculator and HTTP status code reference. Type an IP/CIDR (e.g. `192.168.1.0/24`) for subnet details, or search HTTP status codes by number or description.

See [TOOLS.md](TOOLS.md) for the full list of implemented and planned tools.

## Input Flexibility

- **Case-insensitive** — `KM`, `km`, `Km` all work
- **Full words** — `kilometers`, `pounds`, `degrees fahrenheit`, `fluid ounces`
- **Abbreviations** — `km`, `lb`, `kWh`, `rpm`
- **Ambiguous units show all matches** — `5m` shows both Distance and Duration results
